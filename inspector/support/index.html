PDF Inspector — Support

Welcome to the support page for PDF Inspector. Here you can find information about how to do things in PDF Inspector, or how to get help if you run into issues.
Topics:
  • How to detect link issues in a PDF
  • How to create/delete/modify links in a PDF
  • How to automatically scan a page in a PDF for metadata information

How to detect link issues in a PDF

To detect and/or fix issues with internal links:

  1. Click on Tools > Issues > Links
  2. Verify that the presented issues (if any) are correct
  3. Click on "Resolve Issues" (if any) to resolve the issues
Note that issues are sometimes stacking, which means PDF Inspector will sometimes give a new set of issues and ask you whether to fix them as well. This may happen more than once.

To detect issues with external links

  1. Click on Tools > Issues > External Links
  2. A progress bar appears and, depending on how many links there are in the PDF, may take a while to complete.
  3. A window with information about problems encountered, if any, appears. At the top of the window is a summary, indicating total number of URLs, how many URLs were okay, okay (with irregular status codes), and/or gave errors (such as a 404 error).

How to create/delete/modify links in a PDF

Creating new links

  1. To create a new internal or external link, do: Tools > New Internal Link or Tools > New External Link
  2. The Link Properties panel will appear if hidden, and a new click area will appear on the PDF as a red square
  3. Drag the red square around to move it; drag at the corners or edges to resize it (note: if you have a report window open, you sometimes need to close it in order to be able to move the square)

Deleting a link

  1. Select the link by clicking on it (if you don't see a colored rectangle around the link, you may need to enable Highlight Links)
  2. In the Link Properties panel, click on "Del"

Modifying a link's properties

  1. Select the link by clicking on it (if you don't see a colored rectangle around the link, you may need to enable Highlight Links)
  2. In the Link properties panel, click on "Edit" to begin editing
  3. Click on the property you wish to edit (you may need to double click). Note that you cannot change certain properties, such as Source page or Object ID.
  4. (optional) Click "Done"

How to automatically scan a page in a PDF for metadata information

  1. Go to the page containing metadata information
  2. Click on Tools > Metadata Scan Page
  3. Review the resulting metadata information and adjust misinterpretations where appropriate
